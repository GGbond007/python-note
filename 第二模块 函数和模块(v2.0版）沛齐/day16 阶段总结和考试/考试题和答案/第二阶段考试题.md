10. 写装饰器

    ```python
    # 写timer装饰器实现：计算fun函数执行时间，并将结果给 result，最终打印（不必使用datetime,使用time.time即可）。
    
    @timer
    def func():
        pass
    
    result = func()
    print(result) 
    ---------------------------------------------------
    import time
    def timer(func):
        def inner():
            start_time = time.time()
            func()
            stop_time = time.time()
            return stop_time - start_time
        return inner
    
    @timer
    def func():
        time.sleep(1)
    
    result = func()
    print(result)
    ```
